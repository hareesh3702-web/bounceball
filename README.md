# Classic Bounce Quest

Classic Bounce Quest is a browser-based game inspired by the classic Nokia Bounce look and feel, built only with HTML5, CSS3, JavaScript (ES6), and the Canvas API.

## Highlights

- Glossy red bouncing ball with squash, stretch, rolling, and momentum
- Light blue sky, orange brick platforms, golden rings, and retro-style HUD
- 25 JSON-based levels for GitHub Pages or direct browser use
- Desktop and mobile controls
- Coins, keys, doors, rings, flags, teleporters, enemies, and power-ups
- Procedural sound effects and lightweight background music
- LocalStorage progress save

## Run

Open `index.html` directly, or use:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Controls

- Left Arrow: Move left
- Right Arrow: Move right
- Up Arrow / Space: Jump
- Mobile: on-screen buttons

## GitHub upload

Upload all files so the repo root contains:

- `index.html`
- `style.css`
- `script.js`
- `assets/`
- `README.md`
- `LICENSE`

Do not keep them inside another nested folder unless your Pages workflow publishes that subfolder.
